# Code Folder

This folder contains all your code files and projects.

## Purpose
Store source code files across different programming languages.

## Supported File Types
- JavaScript (.js, .jsx)
- TypeScript (.ts, .tsx)
- Java (.java)
- Python (.py)
- C/C++ (.c, .cpp, .h)
- Web (HTML, CSS)
- Configuration (JSON)

## Best Practices
1. Organize code by project or language
2. Use meaningful file names
3. Add comments and documentation
4. Keep files focused on single responsibility
